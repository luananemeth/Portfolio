# 💻 Projeto Front-End – Fundamentos de HTML e CSS

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de consolidar os fundamentos de **HTML e CSS**, aplicando boas práticas de estruturação, organização e estilização de páginas web.

Durante o desenvolvimento, foram explorados conceitos essenciais para construção de interfaces modernas, com foco em organização semântica, reutilização de classes, estilização de componentes e posicionamento eficiente de elementos.

---

## 🚀 Conceitos e práticas aplicadas

### 🧱 Estrutura e organização

* Criação de **sections** no HTML
* Uso estratégico de **divs**
* Hierarquia entre elementos pai e filho
* Indentação e organização do código
* Boas práticas na nomeação de classes e seções
* Criação da estrutura de **header** e **footer** da página
* Implementação de navegação utilizando a tag `<nav>`
* Desenvolvimento de navegação entre as páginas **Home** e **Sobre mim**

---

### 🎨 Estilização com CSS

* Utilização de classes no HTML e CSS
* Diferentes seletores CSS
* Reset de estilos padrão do navegador
* Aplicação do **Box Model** (`margin`, `border` e `padding`)
* Uso de `box-sizing`
* Controle de `height`
* Unidade `viewport` (vh, vw)
* Utilização de **variáveis CSS** para padronização de estilos
* Customização visual do projeto através de variáveis CSS

---

### 📐 Posicionamento e Layout

* Posicionamento com **Flexbox**
* Uso de `justify-content` para alinhamento de elementos
* Organização de elementos em coluna
* Utilização de `gap` para espaçamento entre elementos filhos
* Definição de espaçamentos verticais e horizontais

---

### 🔤 Tipografia

* Alteração de `font-size`, `font-family` e peso da fonte
* Importação de fontes utilizando **Google Fonts**

---

### 🖱 Componentes e interações

* Estilização de botões e links clicáveis
* Adição de **ícones em links interativos**
* Aplicação de efeitos visuais com o seletor `:hover`
* Remoção de sublinhado padrão com `text-decoration`
* Arredondamento de bordas com `border-radius`

---

### 🛠 Ferramentas utilizadas

* HTML5
* CSS3
* Figma (para consulta de layout, medidas e espaçamentos)
* DevTools do navegador para inspeção e testes de layout
